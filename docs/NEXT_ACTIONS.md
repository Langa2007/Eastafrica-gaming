# Next Actions

This is the immediate task list for starting the basic phase.

## Decision 1: Prototype Engine

Default recommendation:

- Use Godot 4 for the first prototype if the current machine is modest.

Alternative:

- Use Unreal Engine 5 only if the laptop has enough RAM, dedicated GPU memory, and storage.

## Decision 2: First Map

Recommended first map:

- Fictional Nairobi logistics test area.

Map ingredients:

- Depot.
- Paved road.
- Dirt road.
- Hill.
- Tight turn.
- Delivery yard.

## Decision 3: First Truck

Recommended first truck:

- Generic medium box truck.

Why:

- Easier to control than a full articulated truck.
- Still gives a clear trucking feel.
- Lets us test cargo, fuel, and delivery systems quickly.

## First Build Tasks

1. Create engine project.
2. Add placeholder truck.
3. Add test road.
4. Add camera follow.
5. Add depot trigger.
6. Add delivery trigger.
7. Add job board.
8. Add money reward.
9. Add fuel drain.
10. Add cargo condition.

## First Playtest Questions

Ask these after the first playable loop:

- Is the truck enjoyable to drive?
- Is the route clear without explanation?
- Does fuel or cargo damage add useful tension?
- Does the job board make you want to take another job?
- What felt African or East African, even with placeholder assets?
